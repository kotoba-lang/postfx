# ADR 0001: post-effect configuration is sovereign Kotoba

`src/postfx.kotoba` is the sole production language source. It retains all
effect constructors, pipeline composition, curated presets, and uniform-layout
configuration as canonical bounded documents. JVM Clojure is a compiler/test
host only.

Each pipeline is capped by the canonical document-vector budget of 32 effects.
Host objects are not accepted as configuration, and persistent updates cannot
mutate caller-owned data.

This package describes passes only. Shader execution, uniform upload, WebGPU,
render-pass scheduling, and device access remain explicit Kami providers.

Reference execution, restricted JavaScript, and instantiated typed Wasm must
agree on observable values, typed ABI, effects, bounds, and rejection behavior.
Generated byte identity is not required.
